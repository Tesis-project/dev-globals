Objetivo del manual El uso de este manual tiene como objetivo proporcionar una guía a futuros desarrolladores en la implementación, actualización y mantenimiento del sistema. Requerimientos -Sistema Operativo Windows (10 o superior) o GNU LINUX. -Visual Studio Code o cualquier otro editor de código. -Navegador Web (Chrome o Mozilla Firefox) capaz de ejecutar Javascript V8 en adelante. -Insomnia o Postman para realizar pruebas de API. -NodeJS minimo version 18. -pnpm habilitado en nodejs -Docker desktop -HeidiSQL o cualquier otro gestor de base de datos PostgreSQL -CMD, PowerShell o terminal de linux REQUERIMIENTOS DEL CLIENTE 1. Navegador web (Chrome o Mozilla Firefox) capaz de ejecutar Javascript V8 en adelante. Instalación y configuración: Frontend 1. Descargar o clonar el repositorio de GitHub en la siguiente dirección: https://github.com/alvarosego01/thesis_frontend/tree/dev 2. Abrir la carpeta del proyecto en Visual Studio Code o cualquier otro editor de código. 3. Abrir una terminal en la carpeta del proyecto y ejecutar el comando `pnpm install o npm install` para instalar las dependencias necesarias. 4. Una vez instaladas las dependencias, ejecutar el comando `pnpm dev o npm run dev` para iniciar el servidor de desarrollo. 5. Abrir el navegador web y acceder a la dirección http://localhost:5173 para visualizar la aplicación. Backend 1. Descargar o clonar el repositorio de GitHub en la siguiente dirección: https://github.com/Tesis-project/launcher/tree/dev 2. Abrir la carpeta del proyecto en Visual Studio Code o cualquier otro editor de código. 3. Abrir una terminal en la carpeta del proyecto y ejecutar: git submodule update --init --recursive 4. Una vez instaladas las dependencias, ejecutar el comando `pnpm install o npm install` para instalar las dependencias necesarias.
